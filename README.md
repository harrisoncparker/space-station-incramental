# OUTPOST ZERO

A text-based incremental space station manager built in React.

**Play it:** https://harrisoncparker.github.io/space-station-incramental/

---

Inspired by *A Dark Room*. Monospace terminal aesthetic. No icons, no graphics — pure text and numbers.

Manage resources, assign crew to roles, launch surface missions, handle dock events, and research upgrades to keep your orbital outpost alive long enough to activate the Warp Beacon.

## Running locally

```bash
npm install
npm run dev
```

## Tech

- React with `useReducer` for game state and `useEffect` for the 1-second tick
- Vite
- No dependencies beyond React
